# Sri Lanka Trip Guide

Personal travel companion app for Sri Lanka, Sep 20 – Oct 2 2026.

Built as a Progressive Web App — open in a browser and tap "Add to Home Screen" to install.

## Files

- `index.html` — the app
- `manifest.json` — PWA manifest
- `sw.js` — service worker for offline support
- `icon-192.png` / `icon-512.png` — home screen icons

## Updating

Replace `index.html` with the new version and push. GitHub Pages will update within a minute.
